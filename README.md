# Banking Transactions & Accounts – Power BI Dashboard

🚀 Project Overview
Interactive Power BI dashboards analyzing multi‑currency banking transactions and customer accounts. The solution combines a clean star‑schema model with business KPIs to show how money flows through accounts and how the portfolio is distributed across products, sectors and risk segments.

✨ Key Features
Transactions page with total transaction amount, transaction count and distinct accounts, broken down by transaction type, month and city.

Dynamic currency handling: original transaction currency stored in the model and converted into a common reporting currency for analysis.

Accounts page with total and average balance, total loan amount, average interest rate and credit score by account type, sector and city.

Residence‑time and credit‑score banding to highlight how balances and loan exposure are concentrated in different customer risk segments.

Consistent KPI cards and slicers across pages (year, account type, sector, city, residence band, credit score band) for a clean executive view.

🛠️ Tech Stack
Power BI Desktop – data model, DAX measures, KPI cards and interactive dashboards.

Power Query – data cleaning, date transformations, currency codes unification and creation of banded dimensions.

Excel / CSV – source files for synthetic banking transactions and account data, used for validation and quick checks.
